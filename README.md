> A sleek and intuitive web application for managing student records with advanced features like search, sort, statistics, and data export. Built with modern web technologies for a seamless user experience.

## ✨ Features

- **➕ Add Students**: Easily add students with name, age, and marks (with built-in validation)
- **🔄 Generate Random**: Quickly populate the list with randomly generated student data
- **🔍 Search Functionality**: Real-time search by student name
- **📊 Sorting Options**: Sort students by marks (ascending/descending) or alphabetically by name
- **📈 Class Statistics**: View overall percentage, highest and lowest marks
- **🗑️ Delete Students**: Remove students from the list with a single click
- **📄 Export Data**: Export student data to JSON format
- **🏆 Grade Calculation**: Automatic grade assignment based on marks (A, B, C, F)
- **📱 Responsive Design**: Works beautifully on desktop and mobile devices

## 🚀 Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with gradients and modern UI elements
- **Fonts**: Google Fonts (Poppins)
- **Icons**: Built-in CSS styling

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/smart-student-manager.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd smart-student-manager
   ```

3. **Open in browser**:
   - Simply open `index.html` in your preferred web browser
   - No server required - it's a static web application!

## 🎯 Usage

1. **Add a Student**:
   - Enter the student's name, age (18-60), and marks (300-1100)
   - Click "Add Student" to add them to the list

2. **Generate Random Student**:
   - Click "Generate Random" to add a randomly generated student

3. **Search Students**:
   - Type in the search box to filter students by name

4. **Sort Students**:
   - Click "Sort by Marks" to toggle between ascending/descending order
   - Click "Sort by Name" to sort alphabetically

5. **View Statistics**:
   - Check the "Class Statistics" section for overall performance metrics

6. **Delete a Student**:
   - Click the "X" button next to any student to remove them

7. **Export Data**:
   - Click "Export JSON" to export the student list (check console for output)
